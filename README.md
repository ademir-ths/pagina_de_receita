🍳 Cupcake de café com chantilly
Uma página web simples e responsiva que apresenta uma receita deliciosa, desenvolvida para praticar fundamentos de HTML e CSS.

🔗 Acesse o Projeto Aqui
https://github.com/ademir-ths

🚀 Tecnologias Utilizadas
HTML5: Estruturação semântica do conteúdo.
CSS3: Estilização, layout e responsividade.
Google Fonts: Para uma tipografia moderna e legível.

🛠️ O que eu pratiquei:
Uso de Tags Semânticas (main, section, article).
Flexbox/Grid para alinhar ingredientes e instruções.
Media Queries para garantir que a receita fique bonita no celular e no desktop.
Estilização de listas ordenadas e não ordenadas.
